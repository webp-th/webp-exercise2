< Introduction of HTML elements >

このサイトでは、たくさん存在するhtml要素の中から3つピックアップして紹介しています．
また、HTML ( WHATWG Living Standard ) の「 4 The elements of HTML 」を参考にしました．
( 参考：HTML、WHATWG Living Standard、https://html.spec.whatwg.org/multipage/#toc-semantics 、閲覧日：2021/10/31、最終更新日：2021/10/29 )

    < 目次 >
  1. strong要素
  2. li要素
  3. span要素
  1. strong要素
    2. strong要素は、記述しているコンテンツを強調するために使われるものです．strong要素を用いる例を次に示します．
    <index.html> 
      
                <!DOCTYPE html>
    
                <html>
    
                  <head>
    
                   <meta charset="utf-8">
    
                   <title>strong要素の例</title>
    
                    </head>
    
                  <body>
    
                   <p>strong要素は使われていません<br><strong>strong要素が使われています</strong></p>
    
                  </body>
    
                </html>
    実行例

    <index.html> 
      
                <!DOCTYPE html>
    
                <html>
    
                  <head>
    
                   <meta charset="utf-8">
    
                   <title>strong要素の例</title>
    
                    </head>
    
                  <body>
    
                   <p>数学の課題の提出期限は、<strong>1月1日</strong>です．</p>
    
                  </body>
    
                </html>

    上の画像のようにstrong要素で強調したい部分を括ると、文字が太字になります．このようにすることで、webページ内の重要な部分を目立たせることができます．
  2. li要素
    3. li要素は、リストの項目を表すために使われるものです．ol,ul,menu要素を親要素として用いられます．li要素を用いる例を次に示します．

    ol要素を親要素としたとき

    <index.html> 
      
                <!DOCTYPE html>
    
                <html>
    
                  <head>
    
                    <meta charset="utf-8">
    
                    <title>li要素の例</title>
    
                  </head>
    
                  <body>
    
                    <div>
    
                      ol要素を親要素としたとき
    
                      <ol>
    
                        <li>りんご</li>
    
                        <li>バナナ</li>
    
                        <li>メロン</li>
    
                      </ol>
    
                    </div>
    
                  </body>
    
                </html>
    実行例

    ol要素を親要素にすると、リストの項目に番号がつきます．

    ul要素を親要素としたとき

    <index.html> 
      
                <!DOCTYPE html>
    
                <html>
    
                  <head>
    
                    <meta charset="utf-8">
    
                    <title>li要素の例</title>
    
                  </head>
    
                  <body>
    
                    <div>
    
                      ul要素を親要素としたとき
    
                      <ul>
    
                        <li>自転車</li>
    
                        <li>バイク</li>
    
                        <li>メロン</li>
    
                      </ul>
    
                    </div>
    
                  </body>
    
                </html>
    実行例

    ul要素を親要素にすると、リストの項目に「・」がつきます．

    menu要素を親要素としたとき

    <index.html> 
      
                <!DOCTYPE html>
    
                <html>
    
                  <head>
    
                    <meta charset="utf-8">
    
                    <title>li要素の例</title>
    
                  </head>
    
                  <body>
    
                    <div>
    
                      menu要素を親要素としたとき
    
                      <menu>
    
                        <li>朝</li>
    
                        <li>昼</li>
    
                        <li>夜</li>
    
                      </menu>
    
                    </div>
    
                  </body>
    
                </html>
    実行例

    menu要素を親要素にすると、リストの項目に「・」がつきます．

    数字や点を消したとき

    <index.html> 
      
                <!DOCTYPE html>
    
                <html>
    
                  <head>
    
                    <meta charset="utf-8">
    
                    <title>li要素の例</title>
    
                  </head>
    
                  <body>
    
                    <div>
    
                      数字や点を消したとき
    
                      <ol style = "list-style: none;">
    
                        <li>トラ</li>
    
                        <li>ライオン</li>
    
                        <li>イノシシ</li>
    
                      </ol>
    
                    </div>
    
                  </body>
    
                </html>
    実行例

    上のプログラムのようにタグ内に「list-style: none;」を書き加える、もしくは、cssで「list-style: none;」を書き加えることで、項目番号や「・」を表示させなくすることができます．
  3. span要素
    4. span要素は、classやlang、dirなどのグローバル属性と共に使用します．そのため、span要素自体は特に意味はありません．
    <index.html> 
      
                <!DOCTYPE html>
    
                <html>
    
                  <head>
    
                    <meta charset="utf-8">
    
                    <title>span要素の例</title>
    
                  </head>
    
                  <body>
    
                    <p>
    
                     <span style = "background-color: bisque;">こんにちは</span>
    
                    </p>
    
                  </body>
    
                </html>
    実行例

    <index.html> 
      
                <!DOCTYPE html>
    
                <html>
    
                  <head>
    
                   <meta charset="utf-8">
    
                   <title>span要素の例</title>
    
                  </head>
    
                  <body>
    
                    <ol>
    
                      <span　dir = "ltr">
    
                        <li>ラーメン</li>
    
                        <li>うどん</li>
    
                      </span>
    
                    </ol>
    
                  </body>
    
                </html>
    実行例

    今回は、style属性とdir属性を用いてスタイリングと文字列の方向を指定しました．
